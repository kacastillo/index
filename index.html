Introduction to HTML

What is HTML and its role? HTML is a set of code that is inserted on a file to be displayed on the Internet. An examle of this would be a website.

Why use Semantic HTML elements vs. divs? Semantics are easier to read and helps the structure of HTML be consistent and neat.

Why learn HTML? It is a beneficial coding language to help with creating websites and one of the key benefits for becoming a software engineer.
